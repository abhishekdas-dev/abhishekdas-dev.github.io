This project implements an automatic rain sensing car wiper system using the AT89C51 microcontroller. The system uses a raindrop sensor to detect rain intensity and automatically controls the wiper speed using a servo motor.

## Key Features

- **Automatic Rain Detection**: Uses raindrop sensor to detect precipitation
- **Variable Speed Control**: Adjusts wiper speed based on rain intensity
- **8051 Microcontroller**: Built on AT89C51 platform
- **Servo Motor Control**: Precise wiper movement control

## Components Used

- AT89C51 Microcontroller
- Raindrop Sensor
- Servo Motor
- LCD Display
- Power Supply Unit
