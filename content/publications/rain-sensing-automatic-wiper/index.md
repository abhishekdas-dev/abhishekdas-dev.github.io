An automatic car wiper system using embedded microcontroller technology for enhanced driver safety during rainy conditions.

## Key Features

- Rain detection using capacitive rain sensor
- AT89C51 microcontroller-based control system
- Variable speed control based on rain intensity
- Automatic activation and deactivation
