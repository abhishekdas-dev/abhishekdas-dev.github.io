A smart refrigerator system leveraging IoT technology to simplify household tasks with intelligent monitoring and user-friendly mobile interface.

## Key Features

- Real-time inventory tracking using load cells and sensors
- Mobile alerts for restocking when items run low
- Recipe suggestions based on available ingredients
- Android application for remote monitoring and control
