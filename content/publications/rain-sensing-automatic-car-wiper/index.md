This project implements an automatic rain sensing car wiper system using the AT89C51 microcontroller with raindrop sensor for automatic wiper speed control.

## Key Features

- Automatic rain detection using raindrop sensor
- Variable speed control based on rain intensity
- 8051-based AT89C51 microcontroller platform
- Servo motor for precise wiper movement control
